# 基金助手Web端 - 测试指南

## 🧪 功能测试清单

### 1. 启动应用

```bash
cd web
npm run dev
```

浏览器会自动打开 `http://localhost:5173`

---

### 2. 首页功能测试

#### 2.1 添加基金
1. 点击"添加基金"按钮
2. 输入基金代码（测试代码）：
   - `001186` - 富国文体健康
   - `110022` - 易方达消费行业
   - `161725` - 招商中证白酒
3. 输入持有份额：`1000`
4. 输入成本价：`2.5`
5. 选择分组（可选）
6. 点击"确定"

**预期结果**：
- ✅ 基金添加成功
- ✅ 列表中显示新添加的基金
- ✅ 自动获取基金名称和实时数据
- ✅ 计算显示预计收益、持有收益率等

#### 2.2 编辑基金
1. 点击基金行的"编辑"按钮
2. 修改持有份额或成本价
3. 点击"保存"

**预期结果**：
- ✅ 基金信息更新成功
- ✅ 收益数据重新计算
- ✅ 数据保存到本地存储

#### 2.3 删除基金
1. 点击基金行的"删除"按钮
2. 确认删除

**预期结果**：
- ✅ 显示确认对话框
- ✅ 基金从列表中移除
- ✅ 资产总值重新计算

#### 2.4 搜索基金
1. 在搜索框输入基金代码或名称
2. 观察列表变化

**预期结果**：
- ✅ 列表实时过滤
- ✅ 显示匹配的基金
- ✅ 清空搜索框恢复全部显示

#### 2.5 排序功能
1. 点击表格列头（如"预计收益"）
2. 再次点击切换升序/降序

**预期结果**：
- ✅ 列表按选中列排序
- ✅ 显示排序图标
- ✅ 排序状态保存

### 4. 资产展示测试

#### 4.1 资产计算
添加多个基金后，观察顶部资产卡片：

**预期结果**：
- ✅ 资产总值 = ∑(当前净值 × 持有份额)
- ✅ 持有收益 = ∑((当前净值 - 成本价) × 持有份额)
- ✅ 日收益 = ∑(日涨跌额 × 持有份额)
- ✅ 正数显示绿色，负数显示红色

#### 4.2 隐私模式
1. 点击资产卡片区域

## 🐛 已知问题

### 当前版本限制
1. ⚠️ fundgz接口可能有访问限制，建议使用真实基金代码测试
2. ⚠️ JSONBox同步功能尚未实现（第五阶段）
3. ⚠️ 行情中心和基金详情页尚未开发

### 测试建议
1. 使用真实基金代码测试，如：
   - `001186` - 富国文体健康
   - `110022` - 易方达消费行业
   - `161725` - 招商中证白酒
   - `000001` - 华夏成长
   - `163406` - 兴全合润

2. 测试时注意：
   - 基金数据更新时间：交易日 9:30-15:00
   - 非交易时间可能显示上一交易日数据
   - 部分基金可能暂停交易
