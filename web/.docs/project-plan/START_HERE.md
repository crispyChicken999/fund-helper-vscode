# 🚀 基金助手Web端 - 立即开始

## 快速启动（3步）

### 1️⃣ 安装依赖（如果还没安装）
```bash
cd web
npm install
```

### 2️⃣ 启动开发服务器
```bash
npm run dev
```

### 3️⃣ 打开浏览器
浏览器会自动打开 `http://localhost:5173`

---

## 🎯 快速体验

### 第一步：添加基金
1. 点击"添加基金"按钮
2. 输入测试数据：
   - 基金代码：`001186`（富国文体健康）
   - 持有份额：`1000`
   - 成本价：`2.5`
3. 点击"确定"

### 第二步：查看效果
- ✅ 基金自动获取名称和实时数据
- ✅ 顶部显示资产总值和收益
- ✅ 列表显示基金详细信息

### 第三步：体验功能
- 🔍 搜索：输入基金代码或名称
- 📊 排序：点击表格列头排序
- 📁 分组：点击"+"添加分组
- ✏️ 编辑：点击"编辑"按钮修改
- 🔄 刷新：点击"刷新"按钮更新数据

## 💡 使用技巧

### 1. 隐私模式
点击顶部资产卡片，所有数值变为 `****`

### 2. 快速编辑
右键点击分组标签，快速编辑或删除

### 3. 搜索技巧
支持基金代码和名称模糊搜索

### 4. 排序技巧
点击列头排序，再次点击切换升序/降序

### 5. 分组筛选
点击分组标签，只显示该分组的基金

### 6. 查看行情
点击底部"行情"标签，查看市场行情

### 7. 切换分类
在行情页面点击分类标签（A股、H股、美股、其他）

### 8. 查看基金详情
点击首页基金名称，查看详细信息

### 9. 编辑基金
在详情页点击"编辑基金"按钮，修改份额或成本价

### 10. 复制基金代码
在详情页点击"复制代码"按钮

## 🐛 遇到问题？

### 常见问题

**Q: 基金数据不更新？**
A: 点击"刷新"按钮手动刷新，或等待自动刷新（20秒）

**Q: 添加基金失败？**
A: 检查基金代码是否正确（6位数字），确保网络连接正常

**Q: 数据丢失了？**
A: 数据保存在浏览器LocalStorage中，清除浏览器数据会导致丢失

**Q: 页面显示异常？**
A: 刷新页面（F5），或清除浏览器缓存
