## 风险与缓解措施

| 风险 | 等级 | 缓解措施 |
|------|------|--------|
| fundgz接口不稳定 | 中 | 实现重试机制、缓存策略、错误处理 |
| JSONBox服务不可用 | 中 | 实现离线模式、本地缓存、备用方案 |
| 跨域问题难以解决 | 低 | JSONP已验证可行，已有库支持 |
| 性能问题（大基金列表） | 中 | 提前规划虚拟滚动、分页、缓存 |
| 移动端兼容性问题 | 中 | Element Plus已支持移动端，提早测试 |
| 数据冲突场景复杂 | 中 | 明确规则、充分测试、版本控制 |
