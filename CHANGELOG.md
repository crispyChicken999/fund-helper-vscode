# 更新日志 (Changelog)

本文档记录了"基金助手"(Fund Helper) VS Code 扩展的所有重要更改。

## [0.1.2] - 2026-02-28

### Features

- 使用fetch代替axios，优化statusBar的hover显示信息，添加基金总数显示

### Bug Fixes

- 修复基金刷新，部分基金接口没响应，导致基金列表中不显示该基金的问题

## [0.1.1] - 2026-02-27

### Bug Fixes

- 修复命令识别异常的问题

## [0.1.0] - 2026-02-27

### Features

- 🎉 初始版本发布
- 支持添加自选基金，基于基金代码或名称进行模糊搜索
- 基金列表以 TreeView 形式展示在 VSCode 侧边栏，支持展开查看详情
- 基金 item 展示格式为 `(估算收益) 基金名称 涨跌幅%`，根据排序方式动态切换展示内容
- 涨跌图标（🔺红色/🔻绿色/—灰色）直观标识涨跌状态
- 详情项使用 🔴🟢⚪ emoji 指示盈亏方向
- 支持鼠标悬浮查看 Tooltip，展示完整持仓信息，涨红跌绿高亮显示
- TreeView 顶部排序控制栏，悬浮显示 6 个快捷排序按钮，点击循环切换：降序 → 升序 → 默认
- 排序控制栏 Tooltip 显示持仓概览（总持有金额、日收益、持有收益、持有收益率），带涨红跌绿高亮
- 支持按涨跌幅、估算收益、持有额、持有收益、持有收益率进行升序/降序排序
- 正值括号内容高亮，负值和零不高亮，区分直观
- 默认排序下支持拖拽调整基金顺序
- 加仓功能：选择历史净值日期买入，自动计算加权平均成本
- 减仓功能：选择卖出日期参考，校验份额上限
- 修改持仓：直接修改总份额和成本价，提示显示基金名称和代码
- Inline 操作按钮：`+` 加仓、`-` 减仓、`✏` 修改持仓、`🗑` 删除，悬浮即见
- 状态栏显示总日收益和百分比，涨红跌绿，点击刷新数据
- 导入/导出基金列表（JSON 格式），支持合并或覆盖导入
- 基金数据、排序方式均持久化保存到 VSCode 用户全局配置
- 数据刷新间隔可配置（最小 5 秒，默认 30 秒），仅 A 股交易时间内自动刷新
- 空列表时显示欢迎页，提供添加基金和导入的快捷入口
- 使用 `fundgz.1234567.com.cn` JSONP 接口获取实时估值数据
