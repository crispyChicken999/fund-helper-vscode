# 更新日志 (Changelog)

本文档记录了"基金助手"(Fund Helper) VS Code 扩展的所有重要更改。

## [0.3.4] - 2026-05-08

### 新增功能 (Features)

- ✨ **点击基金名称打开详情**：
  - 点击基金列表中的基金名称可直接打开详情页面
  - 鼠标悬停时显示链接样式（颜色变化）
  - 点击时自动关闭 Tooltip（如果有）
  - 提升用户操作便捷性

### 问题修复 (Bug Fixes)

- 🐛 **修复 TreeView 持仓统计数据不一致问题**：
  - **根本原因**：TreeView 使用 `calcDailyGain()` 函数计算日收益，该函数只使用实际净值和实际涨跌幅（`navChgRt`），没有考虑估值
  - **修复方案**：修改 TreeView 的 `_createSummaryFolder()` 和 `_getSummaryDetails()` 方法，使用与 StatusBar 一致的估算收益计算逻辑
  - **计算逻辑**：开市时使用 `(estimatedValue - netValue) * shares` 计算估算收益，休市时为 0
  - **影响范围**：确保 TreeView 持仓统计与 StatusBar 显示的数据完全一致

- 🐛 **修复自动刷新关闭后仍在刷新的问题**：
  - **根本原因**：项目中存在两个独立的自动刷新定时器（TreeView 和 WebviewView），配置变化时只清除了 TreeView 的定时器
  - **修复方案**：在 WebviewView 中添加 `refreshAutoRefresh()` 方法，配置变化时同时重新设置两个定时器
  - **影响范围**：确保关闭自动刷新时所有定时器都被正确清除

### 新增功能 (Features)

- ✨ **实时净值标识**：收盘后~开盘前，如果基金已更新实际净值，在基金名称前显示 ✓ 图标
  - **显示条件**：非开市状态且非休市日，且净值日期已更新到当天
  - **图标样式**：绿色勾选图标，提示"已更新实际净值"
  - **显示时机**：仅在收盘后~开盘前显示，开盘时和中午休息时不显示

### 优化改进 (Improvements)

- 💡 **统一基金数据获取逻辑**：
  - 所有基金（包括普通基金和QDII）都调用 FundMNFInfo 接口获取实际净值
  - 简化了QDII基金的特殊处理逻辑
  - 确保净值数据的准确性和一致性

- 💡 **智能收益计算**：
  - **交易日开盘中**：使用估值计算当日收益
  - **交易日收盘后~次日开盘前**：
    - 净值已更新：使用实际净值计算（更准确）
    - 净值未更新：使用估值计算（保持实时性）
  - **非交易日（休市）**：使用最新实际净值

- 💡 **日期显示优化**：
  - 当日涨幅/收益列第二行显示各基金的净值日期
  - 表头显示最新交易日日期
  - 根据净值更新情况动态显示日期

### 技术优化 (Technical)

- ♻️ **收益计算函数重构**：
  - `calculateDailyGain`：根据市场状态和净值更新情况智能计算
  - `generateFundRow`：优化当日收益计算逻辑
  - `renderGroupTags`：统一分组统计的收益计算
  - `updateAccountSummary`：前端重新计算账户汇总的日收益

- ♻️ **新增辅助函数**：
  - `shouldShowRealValueIcon`：判断是否显示实时净值标识
  - 优化市场状态判断逻辑

## [0.3.3] - 2026-05-07

### 问题修复 (Bug Fixes)

- 🐛 **修复QDII基金估算收益显示为0的问题**：
  - **根本原因**：webview数据提供者中，当QDII基金的`estimatedValue`为null时，错误地fallback到了`netValue`，导致`gsz`等于`dwjz`，计算估算收益时结果为0
  - **修复方案**：将fallback值改为"0"，确保QDII基金走正确的计算分支：`estimatedGain = (holdingAmount * gszzl) / 100`
  - **影响范围**：修复后QDII基金的估算收益能正确显示（如：1026.60 × 2.76% = 28.33）

### 新增功能 (Features)

- ✨ **搜索支持分组标签**：搜索框现在支持搜索分组名称，输入分组关键词可快速筛选该分组下的所有基金

### 优化改进 (Improvements)

- 💡 **QDII基金估算净值推算**：对于QDII基金（gsz为0），通过`单位净值 × (1 + 估算涨跌幅/100)`推算估算净值，tooltip和"成本/最新"列第二行显示推算值
- 💡 **QDII基金更新时间优化**：
  - 表格列：估算涨幅列第二行显示当前时间（格式：HH:mm）
  - Tooltip：更新时间显示当前时间（格式：MM-DD HH:mm）
  - 判断逻辑：检查`fullUpdateTime`是否不包含冒号，如不包含则生成当前时间
- 💡 **QDII基金关联板块智能识别**：根据基金名称自动识别关联板块
  - 包含"港股"/"香港"/"恒生" + "科技" → "港股科技"
  - 包含"港股"/"香港"/"恒生" → "港股"
  - 包含"纳斯达克"/"标普"/"美股" → "美股"
  - 其他QDII基金 → "QDII"
- 💡 **QDII基金"成本/最新"列优化**：第二行显示估算净值而非单位净值，更符合实时估值场景

### 技术优化 (Technical)

- ♻️ **估算数据显示逻辑优化**：新增`shouldShowEstimated`判断条件，对QDII基金（gsz === 0 && gszzl !== 0）即使数据未更新到今天也显示估算数据
- ♻️ **数据源修复**：`src/sidebar/webview/index.ts`中修复QDII基金gsz的赋值逻辑
- ♻️ **搜索逻辑增强**：搜索过滤器新增分组字段匹配，支持基金名称、代码、分组三维度搜索
